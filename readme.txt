WordPress Google Translate makes your website multilingual and available to the world using Google Translate

== Description ==

This plugin allows you to insert the Google Translate widget anywhere on your website using [google-translate] shortcode.

== Installation ==

1. Upload `google-translate` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can add a widget to your website or use [google-translate] syntax inside pages where you want it to appear.

== Frequently Asked Questions ==

== Configuration ==

To configure the widget go to Settings -> Google Translate
